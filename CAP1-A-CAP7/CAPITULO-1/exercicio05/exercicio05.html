EX 05: Qual é o caminho percorrido por uma requisição HTTP a uma página de hipertexto estática?

São esses passos:

  • Requisição HTTP: Quando um usuário solicita uma página web, o navegador envia uma requisição HTTP ao servidor onde a página está hospedada.
  • Processamento pelo Servidor: O servidor processa a requisição e busca o arquivo HTML correspondente.
  • Resposta ao Navegador: O servidor então envia o arquivo HTML de volta ao navegador como resposta à requisição HTTP.
  • Renderização da Página: Finalmente, o navegador renderiza a página HTML para que o usuário possa visualizá-la.
