EX 02: Quantos arquivos CSS e Javascript podem estar vinculados a uma mesma página de hipertexto? Justifique sua resposta.

  • Arquivos CSS: Não há um limite específico para a quantidade de arquivos CSS que podem ser vinculados a uma página HTML. A prática recomendada é minimizar o número de arquivos para otimizar o carregamento da página.
  • Arquivos Javascript: Similarmente aos arquivos CSS, não há um limite definido para a quantidade de arquivos Javascript. Contudo, é importante gerenciar os arquivos de forma eficiente para não afetar o desempenho da página.
