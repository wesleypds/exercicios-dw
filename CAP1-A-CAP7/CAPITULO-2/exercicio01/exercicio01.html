EX 01: Qual é o elemento HTML que contém todos os demais elementos de uma página?

O elemento HTML que contém todos os demais elementos de uma página é o <html>. Este é o elemento raiz de uma página HTML e encapsula todos os outros elementos, incluindo <head> e <body>, que por sua vez contêm informações sobre o documento e o conteúdo da página, respectivamente.
