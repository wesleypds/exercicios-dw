EX 02: Qual é o elemento HTML que contém elementos com metainformações sobre uma página web?

O elemento HTML que contém elementos com metainformações sobre uma página web é o <head>. Este elemento geralmente inclui títulos, links para folhas de estilo (CSS), scripts e outras metainformações como meta tags que definem o conjunto de caracteres, descrição da página, palavras-chave, autor da página e outros dados.
