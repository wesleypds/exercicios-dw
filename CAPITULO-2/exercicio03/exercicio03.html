EX 03: Qual é o elemento HTML que contém todos os elementos de conteúdo de uma página web?

O elemento HTML que contém todos os elementos de conteúdo de uma página web é o <html>. Este é o elemento raiz de uma página HTML e engloba todos os outros elementos, incluindo <head> e <body>, que contêm as informações de cabeçalho e o conteúdo da página, respectivamente.
